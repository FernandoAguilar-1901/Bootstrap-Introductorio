Bootstrap-Intro

Clases principales y de mayor uso con el framework Bootstrap


class="container"

Ocupa las 12 columnas del grid que proporciona CSS, pero deja un margen a los costados

class="container-fluid"

Ocupa el tamaño completo del viewport

class="row"

Indica que el elemento contenedor se comprta como una fila dentro del grid

class=col-n/xx

Dos permite establecer el numero de columnas que queramos ocupe un elemento/componente
Se puede delimitar tanto con numeros como con nomenclaturas predefinidas
Ejemplos: sm, md, lg, xl, xxl.
